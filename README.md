# Green Motors Local Purchase Order

Open `index.html` in Chrome or Microsoft Edge.

Included features:

- System-generated LPO numbering starting at `GMSP/LPO/000261`
- Local AED and international USD purchase modes
- Optional 5% VAT for local purchases; VAT is disabled for international purchases
- Expandable line items and a comprehensive UOM dropdown
- Automatic totals and amount in words
- Quotation reference number
- Typed signatory names and three online signature pads
- Save, edit, password-protected delete, record search, and Print/PDF
- Browser-local records plus optional Google Sheets master-list synchronization

See `SETUP-GOOGLE-SHEETS.md` to connect the app to Google Sheets.

Default delete password: `GM@2026`
